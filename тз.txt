
1. Общие сведения 

Сайт для покупки экологически чистых товаров, в котором пользователи могут ознакомиться с ассортиментом и узнать, как они самостоятельно могут помочь природе. 

2. Дизайн и функционал

2.1. Общее 
Дизайн должен быть в преимущественно зеленых, белых и бежевых цветах. Добавленные элементы обязательно должны отсылать к природе. Дизайн должен быть приятен глазу, не перегружен и понятен.  Логотип и название бренда должны быть на каждой странице сверху и снизу. В самом конце каждой страницы также располагаются контактные данные (mail и номер телефона). 
Сайт должен состоять из 3х страниц - главной (2.2), каталога товаров (2.3) и рекомендации для сохранения экологии (2.4). Должна быть панель с быстрым переключением между всеми страницами.
Быстрая регистрация в качестве пользователя.
Вход по паролю в качестве администратора.

2.2. Главная страница
- Обязательно содержит отдельно фразу  - "Твоя забота делает мир лучше".
- Должна содержать самые популярные товары с перенаправлением на страницу со всем каталогом.
- Также нужно выделить место для перехода на рекомендации для сохранения экологии. Для него нужно добавить крупно фразу - "Помочь может каждый", и более мелко "Советуем начать с этого.." с переходом на страницу с рекомендациями.

2.3. Каталог
- Содержит в себе фотографии всех товаров, их названия, краткое описание и цену и кнопку "подробнее", открывающее небольшое окно с полным составом и сроком годности (при наличии).
- Фильтр по категориям - Для дома, Средства личной гигиены, Одежда. 
- Корзина для выбранных товаров с автоматическим подсчетом итоговой цены.
- Информация о количестве товара.
Для администратора: 
- Возможность добавления товаров.
- Возможность устанавливать рекламу.
- Возможность редактировать количество товара.

2.4 Рекомендации 
Содержит в себе несколько простых советов с пояснением к каждому. При возможности добавление картинок с подходящей советам тематикой. 

3. Контент
Будет рассчитан на популяризацию бережного обращения к природе. В будущем планируется переделать "рекомендации" под "статьи", где будет доступно добавление новых статей с советами или новостями.

4. Технические требования
- Использование современных технологий (HTML, CSS, JavaScript)
- Оптимальные алгоритмы поиска
- Интеграция платежных систем
